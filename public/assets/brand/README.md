# Tantyastorm Brand Assets

Copy this folder to `public/assets/brand/`.

- `logo-color.png` — main full-color logo
- `logo-black.png` — black monochrome logo
- `logo-white.png` — white monochrome logo
- `app-icon.png` — main square icon
- `wordmark.png` — wide Tantyastorm logo
- `social-banner.png` — original social banner
- `og-image-1200x630.png` — Open Graph image
- `twitter-card-1200x600.png` — X/Twitter card
- `favicon.ico` — browser favicon
- `apple-touch-icon.png` — Apple touch icon
- `pwa-icon-192.png` and `pwa-icon-512.png` — PWA icons
